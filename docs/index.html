Everything, configurable

A whole personality, under your control.

Persistent memory

Rolling summaries, semantic recall, durable facts, and a server-wide search index — ask "where was that posted?" and get a jump-link.

Teachable knowledge

Feed it web pages, whole sites, or documents. It chunks, embeds, and answers from them in its own voice — plus a glossary for your dialect.

Role-based access

Gate it by Discord role, with live permission re-checks — lose Manage Server and console access is revoked on the next request.

Extensions

Toggle packaged features — dice, calculator, concise mode, and a full Star Citizen pack with live trade, ship and location tools.

Remote access

Let co-admins sign in from anywhere over Tailscale Funnel — a free, stable …ts.net link with no domain and no port-forwarding.

Updates itself

The app checks GitHub Releases and installs new versions in place — one click, no reinstall, on both macOS and Windows.

Secure by design

Your bot. Your keys.
Your machine.

There is no Olisar cloud. You run one app on a machine you own; it brings its own Discord bot and your own free API keys, and everything it knows lives in a single local database. Admins sign in with Discord — only those who should can reach the console.

Discord-OAuth login, admins only Live permission re-checks all data stays local /forget-me for every member free Gemini tier — no paid APIs
Up and running in minutes

No server. No terminal.

Install the app

Download the .dmg or .exe and open it. It bundles the bot, the API, and the dashboard.

Run the wizard

Paste your Discord bot token and a free Gemini key. It validates them and shows the redirect to register.

Sign in & tune

Open the console with the account that has Manage Server, shape its persona, and let it loose.
